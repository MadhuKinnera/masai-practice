b tag

To bold the text in HTML, use either the strong tag or the b (bold) tag. Browsers will bold the text inside both of these tags the same, but the strong tag indicates that the text is of particular importance or urgency. You can also bold text with the CSS font-weight property set to “bold.”

i tag

The tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic. The i tag is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc.

u tag

The u HTML element represents a span of inline text which should be rendered in a way that indicates that it has a non-textual annotation. This is rendered by default as a simple solid underline, but may be altered using CSS.

select tag

The select HTML element represents a control that provides a menu of options.
